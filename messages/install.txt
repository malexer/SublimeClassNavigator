Class Navigator
~~~~~~~~~~~~~~~

Class Navigator helps to quickly jump between classes and
functions/methods in a current file.


Quick start
```````````

* Go to Class
    - Linux/Windows: Ctrl + Shift + C
    - macOS: ⌘ + Shift + C

* Go to previous Function
    - Linux: Super + Alt + ↑
    - macOS: ⌘ + Alt + ↑
    - Windows: Win + Alt + ↑

* Go to next Function
    - Linux: Super + Alt + ↓
    - macOS: ⌘ + Alt + ↓
    - Windows: Win + Alt + ↓
