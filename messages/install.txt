Class Navigator helps to quickly jump between classes and
functions/methods in a current file.


Quick start
~~~~~~~~~~~

* Go to Class
    - Linux: Super + Shift + C
    - macOS: ⌘ + Shift + C
    - Windows: Win + Shift + C

* Go to previous Function
    - Linux: Super + Alt + ↑
    - macOS: ⌘ + Alt + Shift + ↑
    - Windows: Win + Alt + ↑

* Go to next Function
    - Linux: Super + Alt + ↓
    - macOS: ⌘ + Alt + Shift + ↓
    - Windows: Win + Alt + ↓
